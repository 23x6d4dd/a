Privacy policy for Perpro applications

This privacy policy governs your use of the software application „UTMP Basic“ (later referred as the/this „Application“) for Android™ mobile devices that was created by Perpro.

The Application is an Android™ application intended to read out values measured by USB Digital Thermometer.

User provided information

The Application does not require any user registration. The Application does not require the user to provide any personal information. The Application does not need any personal data.

The only user provided data that this Application uses is a unit of temperature value (Celsius, Fahrenheit).

Automatically collected information

The Application does not collect any information automatically. The application does not need any uplink to any network. The Application does not share any information automatically with third parties. The Application does not collect information about the location of host mobile device.

Automatically accessed information

This application does not need access to any data stored on a host device, except user locale to correctly format read out values.
