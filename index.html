Android Application Development

UTMP Basic

Displays value of a connected USB Thermometer

APK download: v 0.2.0

Supported devices

  • All usbtemp USB Thermometers

Prerequisites

Since most newer Android powered devices have USB type-C connector an OTG adapter cable to USB type-A is needed. USB version 2.0 of the adapter cable is sufficient. Any connected device draws power from Android device, so its battery will be drained a little sooner. Device support of USB host mode is mandatory, device that does not meet this requirement cannot download the application from Google Play.

Startup

After the thermometer is connected to an Android powered device and the application is already installed a notification is displayed saying a new device has been connected. After notification confirmation UTMP Basic application is started.
