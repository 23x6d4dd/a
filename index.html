Android Application Development

UTMP Basic

Displays value of a connected USB Thermometer

Supported devices

  • All usbtemp USB Thermometers

Instructions

Since most newer mobile phones have USB type-C connector an OTG adapter cable to USB type-A is needed. USB version 2.0 is sufficient. Any connected device draws power from Android device, so its battery will be drained a little sooner.

After the thermometer is connected to an Android powered phone a pop-up is displayed asking for user-permission whether to allow UTMP Basic to handle the thermometer. When approved UTMP Basic is started and values are displayed.

At the start of the application a list of detected devices is displayed. Device is selected by taping on it.

When value is being displayed an unit (Celsius or Fahrenheit) could be selected in the menu upper-right.
